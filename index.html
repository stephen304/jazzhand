A new revolution in user input technology

Fully configurably, and seamlessly binds a Leap Motion controller to any part of your system.

Dependencies

You must have the LeapMotion SDK installed and avaliable on your system to build jazzhand. Additionally, xdotool is required to be present on the system


$ git clone http://github.com/Stephen304/jazzhand
$ cd jazzhand
$ EDIT config.h
$ make
$ ./jazzhand
